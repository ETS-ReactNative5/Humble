What is Humble Humble is a framework unlike any other in the PHP world. By leveraging a "less is more" philosophy with powerful conventions, a polyglot ORM, and a graphical workflow editor, it is possible to create amazing and modular applications that can function like traditional web apps, or newer decoupled UI apps. The choice is yours.
Dependencies The following products need to be installed and properly configured prior to attempting installl

  • Apache 2.4*
    • Enable Rewrite Module
    • Enable SSL Module
  • PHP 7.1+
    • Add MongoDB Extension
    • Add Memcache Extension
    • Enable Mysqli Extension
    • Enable Soap Extension
  • PHP Composer
  • MySQL 5.6+
  • MongoDB 3.4+
  • Optional [optimization]
    • Memcached
  • Optional [support for code deployments]
    • Java JDK 1.8
    • Apache Ant 1.8+
  • Optional [support for developers]
    • Node.js
    • Grunt
* Your website must be configured and running before the installation takes place.

Windows Installation,Getting Started First you will need to create the directory where you plan on installing Humble.

You will then need to download the initial package installer into that directory
  • Humble.php
  • Humble.bat (optional Windows helper batch file)

Now you are ready to create a Humble project, run humble --project to create the new project. This will guide the install process and will identify the source for any future updates of the framework.

Finally, after the project is created you are ready to fetch and install the source. Run humble --fetch, and you will be taken to the projects website to provide the run time configuration information.

For older installations Below is the patching file you will need to bring your earlier alpha versions of Humble up to the current version. Only do this if you have a current installation of Humble. Right click and download the file below into your projects /app directory.

Then from the /app directory, run humble --patch

  • Module.php Patching file


Linux Installation, Getting Started First you will need to create the directory where you plan on installing Humble.

You now need to download the initial package installer into that directory
  • Humble.php
  • humble.sh(option shell helper file)

Now you are ready to create a Humble project, run php Humble.php --project to create the new project. This will guide the install process and any future updates of the framework.

Finally, after the project downloads you are ready to install it. Run php Humble.php --install, you will need to provide answers to all the questions asked
